Real time OS From Scratch (RFS) ‣ GPL3

A real time operating system that runs on cortex-M4 (tivaC) This is inspired from `Linux from scratch` name, but it is not related to it at all. This RTOS contains:

  • Kernel with round-robin preemptive scheduler.
  • FatFs as file manager.
  • Drivers framework like:
  • GPIO, UART, SPI, TIMER, PLL control library
  • Data structures like:
  • Stack, Queue, Priority Q, Inverted Priority Q, Mailbox, LinkedList, Sorted Linked List, Sorted Linked List With ID
  • CMake build system

The whole project is relay on ARM-none-Eabi Toolchain.

Link: https://github.com/RTOS-From-Scratch
